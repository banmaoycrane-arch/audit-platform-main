## 一、序时簿（场景 A）

| 字段 | 优先级 | 说明 |
|------|--------|------|
| 凭证日期 | P0 | |
| 凭证号 | P0 | 不同软件导出列名不同（凭证号/凭证编号） |
| 分录行号 | P1 | 部分软件导出带行号，取决于导出选项 |
| 摘要 | P0 | |
| 科目编码 | P1 | 有则必取 |
| 科目名称 | P0 | |
| 借方金额 | P0 | |
| 贷方金额 | P0 | |
| 往来户名称 / 对方单位 | P1 | **≠ 存货单位/计量单位**（已修正误映射） |

## 三、银行流水（场景 A 为主，B 为扫描/图片）

| 字段 | 优先级 | 说明 |
|------|--------|------|
| 交易日期 | P0 | |
| 借/贷或发生额 | P0 | |
| 余额 | P1 | 有则取 |
| **对方单位** | P0 | 网银 Excel 常见列名（已加别名） |
| 对方户名 | P0 | 与「对方单位」分场景，两者可能只存在其一 |
| 对方银行 / 对方开户行 | P1 | 与户名分开存储 |
| 对方账号 | P1 | |
| 摘要 / 用途 / 附言 | P0 | |

**工程说明**：

- `.xls` 网银导出：已装 `xlrd`，Excel 列映射路径可用  
- 格式 **每年可能改版**：靠别名表 + 修正回流，不追求一次写死  
- 图片/PDF 扫描：场景 B，需 OCR（`easyocr` 已装）+ 必要时 LLM
