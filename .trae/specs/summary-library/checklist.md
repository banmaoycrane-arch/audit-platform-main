# 智能摘要库与逻辑校验系统 - 验收清单

## 审计导向设计

**核心目标**：代替审计实质性程序中的重新计算，实现全量审计，规避抽样审计风险。

**两种工作模式**：
1. **记账模式**：导入原始资料 → AI 自动编制定录 → 生成账套
2. **审计模式（主要）**：导入原始资料 + 被审计单位分录 → 审计测试 → 错误定位 → 审计语言表述
