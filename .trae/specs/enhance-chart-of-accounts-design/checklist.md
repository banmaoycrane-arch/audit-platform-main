# Checklist

- [x] 当前企业无会计科目时，页面明确显示“当前企业尚未设计会计科目”。
- [x] 空状态说明只围绕本企业建账，不出现泛泛的产品介绍或无关说明。
- [x] 空状态提供“开始设计会计科目”入口。
- [x] 科目设计窗口支持科目代码、科目名称、借贷方向、上级科目。
- [x] 科目设计窗口支持科目大类：资产、负债、所有者权益。
- [x] 资产类支持流动资产、非流动资产。
- [x] 负债类支持流动负债、长期负债。
- [x] 所有者权益支持注册资本、资本公积、盈余公积、未分配利润。
- [x] 注册资本、资本公积、盈余公积默认不纳入可分红基础。
- [x] 未分配利润支持设置是否纳入可分红基础。
- [x] 会计科目新增接口能保存并返回科目大类、科目细类、权益细类、可分红标记。
- [x] 科目列表能显示新增分类字段，旧科目缺少字段时仍可正常显示。
- [x] 核算辅导文案具体解释资产流动性、负债偿还期限、权益和可分红口径。
- [x] 会计科目相关后端测试通过。
- [x] 前端类型检查已执行；如失败，已确认是否属于本次会计科目范围。
